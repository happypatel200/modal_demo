# modal_demo

Technologies Used: React JS, CSS, Vite  
Concepts Covered: useState, Conditional Rendering, Component Communication, Modal Popup

# Project Description:
This project demonstrates how to implement a reusable modal popup in React. 
The modal can be opened and closed dynamically using state management (useState).

Installation

Bash
cd MODAL_DEMO
Install the necessary dependencies:

Bash
npm install
Running the Project
To start the development server:

Bash
npm run dev
This will typically start the application on http://localhost:5173 (or another port) and enable hot module reloading for development.
